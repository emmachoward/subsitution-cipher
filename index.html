Ciphering App

Help?

Type in a message and submit, and an encoded version of the message will be generated using a substitution cipher, along with a key for the randomized letters. This can be handy for creating a puzzle for others to solve. Generally speaking, a sentence or two is an ideal length for these. Be sure to provide one of the letter keys as a hint. The first letter listed is the original alphabet, and the the second is the coded version. Have fun!

by Emma Howard
